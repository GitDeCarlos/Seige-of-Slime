In order to run this project folder within Unity:
1. Download Unity Hub (any version)
2. From within Unity Hub, download Unity version 2020.3.22f1
3. Once download is complete, navigate to the Projects tab of Unity Hub
4. Click "Add" from the top right
5. Navigate to this folder and click select.
6. Now the project should be added to the Projects list and from there you can open and run the game.

Run game standalone:
1. Navigate to "FinalBuild folder within this project folder
2. Extract or open "SOS_1.0.0.rar" to any desired folder location
2. Run "Siege of Slime.exe"
